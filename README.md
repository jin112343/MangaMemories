# MangaMemories

MangaMemoriesは、SwiftUIとKotlinを使用して開発する、漫画風のアルバムアプリです。このリポジトリでは、アプリの開発手順や使用技術、参考となる公式ドキュメントについて説明します。

## 目標
- **2週間以内にアプリを完成させる**  
迅速かつ効率的に開発を進め、2週間での完成を目指します。

## 開発管理
- **GitHubで進捗管理**  
GitHubを利用してコードを管理し、進捗状況を確認・共有できるようにします。
